CSS Flexible Box Layout Module のサンプルページ001

内容

  • flexbox というHTMLの要素があるわけではない。
  • CSSの属性として用意されている。

Flexbox の使い方

  1. ある要素に display: flex を指定する（または display: inline-flex）。
  2. flex-direction プロパティで主軸を指定する。
  3. 主軸方向の配置方法を justify-content プロパティで指定する。
  4. 交差軸方向の配置方法を align-content, align-self / align-items プロパティで指定する。
  5. etc.

flex-grow プロパティ

  • flex アイテムが flex コンテナ内のスペースをどれくらい占有するか指定する。
  • flex アイテムに対して指定する。
  • flex アイテム同士での相対的な値を指定する。
  • 初期値は0。スペースを取らない。
  • 全ての flex アイテムの flex-grow 合計値が 1を超えない場合は、全体のスペースを 1としてそれに対してそれぞれの割合の分だけスペースが分配される？
  • 全ての flex アイテムの flex-grow 合計値が 1を超える場合は、それぞれの flex-grow 値の相対的な割合でスペース全体が分配される？

参考

  • CSS Flexible Box Layout Module Level 1
  • flex-grow - CSS | MDN

flex プロパティ

  • flex-grow, flex-shrink, flex-basis を同時に指定することができる。これらはどれも、flexコンテナに対して、それぞれの flex アイテムがどれだけのスペースを使うかを指定するプロパティである。
  • デフォルト値
    • flex-grow: 0 (スペースをとらない)
    • flex-shrink: 1 (スペースを縮小もしない)
    • flex-basis: auto (基本となるスペースを指定しない)
